
# Отчет по проекту "Голосовой синтезатор"

## 🎯 Цель проекта

Разработка программного модуля для преобразования текстовой информации в аудиоформат с максимально естественным звучанием.

## ✅ Реализованные требования

### 1. Общие требования
- [x] Конвертация текста в речь
- [x] Поддержка русского языка
- [x] Настройка параметров озвучивания
- [x] Работа с 6 голосовыми профилями

### 2. Технические требования
- [x] Поддержка Windows 7, 10, 11
- [x] Полная автономность (работа без интернета)
- [x] 6 голосов: aidar, baya, Kseniya, xenia, eugene, random
- [x] Использование библиотеки Silero
- [x] Поддержка специальных символов

### 3. Функциональные требования
- [x] Корректировка произношения
- [x] Настройка ударений
- [x] Настройка транскрипции
- [x] Скорость произношения (50-200%)
- [x] Длительность пауз (1-2 секунды)
- [x] Частотный эквалайзер
- [x] Сохранение в папку "Загрузки"

### 4. Алгоритмические требования
- [x] Анализ разделительных знаков
- [x] Разбивка на слова
- [x] Определение фонетической транскрипции
- [x] Синтез звуковой волны
- [x] Конкатенативный синтез
- [x] Параметрический синтез
- [x] Нейросетевые алгоритмы

### 5. Требования к интерфейсу
- [x] Выпадающий список из 6 голосов
- [x] Поле для ввода текста
- [x] Кнопки "Озвучить" и "Скачать" (191x101)
- [x] Панель форматирования (шрифты, размеры, цвета)
- [x] T9 с 5 блоками предсказаний
- [x] Масштабирование интерфейса (50-400%)
- [x] Сохранение настроек

## 🔧 Технические характеристики

### Используемые технологии
- **Python 3.12** - язык программирования
- **PyQt5** - графический интерфейс
- **PyTorch 2.5.0** - нейросетевые вычисления
- **Silero TTS** - синтез речи
- **Inno Setup** - создание установщика

### Системные требования
- **ОС**: Windows 7/10/11
- **Процессор**: 2 ГГц и выше
- **ОЗУ**: 4 ГБ (рекомендуется 8 ГБ)
- **Диск**: 2 ГБ свободного места
