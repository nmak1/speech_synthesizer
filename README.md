## 🎯 Описание

Voice Synthesizer - это мощный инструмент для синтеза речи на русском языке, разработанный в соответствии с техническим заданием. Программа использует современные нейросетевые алгоритмы библиотеки Silero TTS для достижения максимально естественного звучания.

### Ключевые особенности

* 🚀 **Высокое качество** синтеза благодаря нейросетевым алгоритмам
* 💻 **Полная автономность** - работает без интернета
* 🎨 **Интуитивный интерфейс** с поддержкой форматирования текста
* 🔧 **Гибкие настройки** скорости, пауз и голосовых профилей
* 💾 **Сохранение** синтезированной речи в аудиофайлы

## ✨ Основные возможности

### 🎙️ Синтез речи

* Конвертация текста в речь с естественным звучанием
* Поддержка русского языка
* 6 голосов на выбор:

  * **aidar** - мужской, четкий
  * **baya** - женский, мягкий
  * **kseniya** - женский, выразительный
  * **xenia** - женский, эмоциональный
  * **eugene** - мужской, спокойный
  * **random** - случайный выбор

### 📝 Редактор текста

* Панель форматирования (шрифты, размер, цвет)
* Жирное начертание
* Цветовая палитра
* Поддержка специальных символов

### 🔤 T9 предсказание

* Автоматическое предсказание слов
* 5 блоков с предсказаниями
* Обучение на основе пользовательского ввода
* Работа без интернета

### 🎚️ Настройки синтеза

* Регулировка скорости произношения (50-200%)
* Настройка длительности пауз (1-2 секунды)
* Корректировка произношения
* Настройка ударений
* Частотный эквалайзер

### 💾 Сохранение

* Сохранение синтезированной речи в WAV формат
* Автоматическое сохранение в папку "Загрузки"
* Возможность сохранения с любым именем

### 🔍 Интерфейс

* Масштабирование интерфейса (50-400%)
* Сохранение настроек между сеансами
* Интуитивно понятная навигация
* Современный дизайн

## 📦 Установка

### Вариант 1: Установщик (рекомендуется)

1. Скачайте `VoiceSynthesizer\_Setup.exe`
2. Запустите установщик
3. Следуйте инструкциям мастера установки
4. После установки запустите программу через ярлык на рабочем столе

### Вариант 2: Портативная версия

1. Скачайте `VoiceSynthesizer.exe`
2. Запустите файл напрямую
3. При первом запуске программа скачает необходимые модели (≈150 МБ)

### Вариант 3: Из исходного кода

```bash
# Клонирование репозитория
git clone https://github.com/nmak1/voice-synthesizer.git
cd voice-synthesizer

# Создание виртуального окружения
python -m venv venv
venv\\Scripts\\activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск программы
python main.py

🚀 Использование
Быстрый старт
Выберите голос из выпадающего списка в верхней панели

Введите текст в редактор или используйте T9 для быстрого ввода

Настройте скорость с помощью слайдера (опционально)

Нажмите "Озвучить" для воспроизведения

Нажмите "Скачать" для сохранения аудиофайла

Работа с T9
Нажмите кнопку "T9" для активации (она станет зеленой)

Начните вводить слово - появятся 5 кнопок с предсказаниями

Нажмите на нужное слово для автозаполнения

Словарь T9 автоматически обучается на вашем вводе

Форматирование текста
Шрифт: выберите из выпадающего списка

Размер: выберите значение из списка

Жирный: нажмите кнопку "B"

Цвет: нажмите кнопку 🎨 или выберите готовый цвет

Масштабирование интерфейса   

Используйте слайдер в правом нижнем углу

Диапазон: 50% - 400%

Значение сохраняется между запусками
